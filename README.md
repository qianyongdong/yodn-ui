[yodn-ui](https://yodn-ui.github.io/yodn/)

## 快速启动

- 安装 pnpm

```
npm install pnpm -g
```

- 安装依赖

```
pnpm install
```

- 打包组件库与工具库

```
pnpm run build:yodn-ui
```

- 启动测试项目

```
pnpm run yodn-ui:dev
```

- 启动静态文档站点

```
pnpm run docs:dev
```
