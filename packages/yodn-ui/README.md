[yodn-ui](https://yodn-ui.github.io/yodn/)

## 快速启动

- 全局安装脚手架并创建项目

```
//安装脚手架
npm install create-yodn-ui -g

//创建项目
npm create yodn-ui

//通过命令行输入参数创建成功...
```

- 
- CD工程目录并安装依赖

```
//移动到目录
cd 你的工程

//安装依赖
npm install
```

- 运行项目

```
//运行
npm run dev

//打包
npm run build
```
